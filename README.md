# Student Grade Calculator

A Python application to calculate student grades, GPA, and generate detailed report cards.

## Features
- Calculate grades for multiple subjects
- Compute GPA on a 4.0 scale
- Generate formatted report cards
- Statistical analysis (average, highest, lowest)
- Grade letter assignment (A, B, C, D, F)

## How to Run
```bash
python grade_calculator.py
```

## Example Output
```
===== REPORT CARD =====
Student: John Doe
Mathematics : 92 -> A
Physics     : 85 -> B
Chemistry   : 78 -> C+
GPA: 3.4 / 4.0
======================
```

## Tech Stack
- Python 3
- No external libraries required

## Use Cases
- Teachers managing class results
- Students tracking their own performance
- Academic institutions automating grading
